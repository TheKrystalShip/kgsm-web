# Game Cover Images

This directory should contain game cover images for each blueprint.

## File naming convention

Save your game cover images with filenames matching the blueprint names in lowercase:

- `minecraft.jpg`
- `factorio.jpg`
- `terraria.jpg`
- etc.

## Sources for game cover images

- You can find official game cover art on Steam store pages
- Game publisher websites
- Ensure you have the right to use any images you add here

## Image specifications

- Resolution: 400x600px recommended (2:3 aspect ratio)
- Format: JPG or PNG
- File size: Keep under 200KB for optimal performance
